Projects

  • healthSystem [in-progress]:
    The purpose of this project is to design and build a lightweight and easily modifiable health data system. This design will be modular to promote customization, and avoid large monolithic repositories. Intended features are to include a database schema, ETL tools, PACS imaging database, and automation services. The project stack includes: Python, PostgreSQL

    In-progress Features:
    - healthdbModels: Framework to construct a simple, customizable health database (HealthDB)
    - healthdbETL: ETL and Database Interaction Tools for HealthDB

  • PsychPredict [in-progress]:
    This project was designed to predict incident psychiatric hospitalizations using data sourced from an electronic medical record at Weill Cornell Medicine. The project incorporates SQL queries, transformations, and common machine-learning algorithms. The project stack includes: Python, PostgreSQL
